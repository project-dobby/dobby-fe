# dobby-fe

- 'Dobby' is a Daily, Weekly, Monthly Report System.

### Development Guide

- Webpack 4.x
- React 16.x
- Typescript 3.x


#### Client Side

- `webpack-dev-server`

```bash
# localhost:9001
yarn start
npm run start
```

#### Server Side

- `nodemon`

```bash
# localhost:9000
yarn server
npm run server
```

#### Run both

- you can access 'client side' server as `9001` port,  
  and 'server side' server as `9000` port.

```bash
yarn dev
npm run dev
```

#### lint (tslint)

```bash
yarn lint
npm run lint
```
