# dobby-fe

- 'Dobby' is a Daily, Weekly, Monthly Report System.

### Development Guide

- Webpack 4.x
- React 16.x
- Typescript 3.x

#### Universal Rendering

- Client Side Rendering

```bash
# localhost:9001
yarn start
npm run start
```

- Server Side Rendering

```bash
# localhost:9000
yarn server
npm run server
```

- Run both 

```bash
yarn dev
npm run dev
```

- lint (tslint)

```bash
yarn lint
npm run lint
```
